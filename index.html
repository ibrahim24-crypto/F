Politique de Confidentialité

Dernière mise à jour : 12 juin 2025

1. Introduction

Cette politique explique comment notre application sur Facebook ("nous", "notre") collecte, utilise et protège vos données personnelles lorsque vous utilisez notre application. En utilisant cette application, vous consentez aux pratiques décrites dans cette politique.

2. Données Collectées

Nous collectons les données suivantes :

  • Données fournies par Facebook (selon vos autorisations) : Nom, photo de profil, adresse email
  • Données générées par l'application : Historique d'utilisation, interactions, données techniques (IP, navigateur, appareil)
  • Données partagées volontairement : Contenu que vous publiez via l'application

3. Utilisation des Données

Vos données sont utilisées pour :

  • Fournir les fonctionnalités de l'application
  • Personnaliser votre expérience utilisateur
  • Améliorer notre application
  • Répondre à vos demandes d'assistance
  • Envoyer des notifications (si activées)

4. Partage des Données

Vos données peuvent être partagées avec :

  • Facebook/Meta : Pour l'authentification et les fonctionnalités de base
  • Fournisseurs de services : Hébergeurs et outils d'analyse sous contrat
  • Autorités légales : Lorsque requis par la loi

Nous ne vendons pas vos données personnelles.

5. Sécurité des Données

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles appropriées pour protéger vos données contre tout accès non autorisé ou toute divulgation.

6. Conservation des Données

Vos données sont conservées :

  • Tant que votre compte est actif
  • Jusqu'à 6 mois après suppression du compte
  • Les données anonymisées peuvent être conservées indéfiniment

7. Vos Droits

Conformément au RGPD, vous avez le droit de :

  • Accéder à vos données personnelles
  • Demander leur rectification
  • Demander leur suppression
  • Retirer votre consentement
  • Limiter le traitement
  • Porter réclamation auprès d'une autorité de contrôle
